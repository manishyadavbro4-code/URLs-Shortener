# 🔗 URL Shortener

A modern URL Shortener web application that allows users to convert long URLs into short, shareable links. It provides a simple and efficient way to manage and access links.

## 🚀 Features

- Shorten long URLs instantly
- Generate unique short links
- Redirect users to original URLs
- Copy shortened URLs with one click
- Responsive and user-friendly interface
- Fast and efficient link management

## 🛠️ Tech Stack

- Next.js
- React.js
- Tailwind CSS
- MongoDB
- Node.js
